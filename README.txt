Birthday Surprise — How to use
-------------------------------
1. Replace photos:
   - Put your photos into the 'assets' folder and name them: photo1.jpg, photo2.jpg, photo3.jpg, photo4.jpg, photo5.jpg
   - Or edit index.html to change paths/captions.

2. Add music (optional):
   - Place an mp3 at assets/song.mp3 (or change the src in index.html).

3. Open index.html in your browser to view the surprise page.
   - Click 'Reveal Surprise' to show the big birthday message with hearts & confetti.
   - Use Prev/Next to navigate the slideshow. Music can be played/paused.

4. If you'd like me to embed your actual images into the zip, upload up to 5 pictures here and I'll create a custom version for you.
